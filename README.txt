pwgen - Passwort-Generator
==========================

Das Programm erzeugt Passphrassen nach der 'Diceware'-Methode (dice: würfeln)
(siehe: https://de.wikipedia.org/wiki/Diceware).

Hierbei wird aus einer Wortliste ein langes aber leicht zu merkendes
Passwort erzeugt. Es steht wahlweise eine deutsche und eine englische Wortliste
zur Verfuegung. Die Wortlisten sind entnommen aus:
- https://raw.githubusercontent.com/dys2p/wordlists-de/main/de-1296-v1-diceware.txt
- https://www.eff.org/files/2016/09/08/eff_short_wordlist_1.txt

Die Anzahl der Worte in der Passphrase können variiert werden, ebenso
die Verwendng von Trennzeichen, Sonderzeichen, Zahlen usw..

Das Programm fragt die Optionen vom Benutzer ab. Hierbei sind Standardwerte gesetzt,
die durch eine Eingabe überschrieben werden können.

Die erzeugte Passphrase wird auf dem Bildschirm angezeigt und gleichzeitig in die
Zwischenablage kopiert, so dass sie dort zum Einfügen zur Verfügung steht.

Beispiele:

Deutschsprachige Passphrase mit Standardeinstellungen:

    pumpe_kaufhaus_FROSCH

Mit Zahlen und Sonderzeichen:

    kaffee2336_kegel_PAVIAN!

Englischsprachige Passphrase mit Standardeinstellungen:

    nail_area_POGO

Mit Zahlen und Sonderzeichen:

    cupid792_mud_OAK!

Passwort aus einem Wort mit Grossbuchstaben, Zahl und Sonderzeichen '$':

    Gulasch9944$
